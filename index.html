Pixel UI
Build Clean, Modern UI Interfaces

Pixel UI is a free Lightweight CSS Component Library to help you build Clean, Modern looking websites and Applications Faster.

Installation

The easiest way to get started with PixelUI is just to copy the CDN link and paste it inside the <head> tag in your HTML.
